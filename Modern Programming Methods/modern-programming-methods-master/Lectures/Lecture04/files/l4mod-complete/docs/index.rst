A Gaussian Elimination routine
******************************

l4mod
========================

This module implements Gaussian elimination [1]_ for :obj:`list of lists` objects, along with hand-written matrix multiplication.

See :func:`l4mod.gauss` and :func:`l4mod.matmul` for more information.

.. automodule:: l4mod
  :members: gauss, matmul, zeromat
